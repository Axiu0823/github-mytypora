## 3.2 算术运算符

### 3.2.1 介绍

算术运算符是对数值类型的变量进行运算的，在Java程序中使用的非常多。

### 3.2.2 算术运算符一览

|  运算符  |     运算     |      范例       |     结果     |
| :------: | :----------: | :-------------: | :----------: |
|    +     |     正号     |       +7        |      7       |
|    -     |     符号     |   b = 11 ;-b    |     -11      |
|    +     |      加      |      9 + 9      |      18      |
|    -     |      减      |     10 - 8      |      2       |
|    *     |      乘      |      7 * 8      |      56      |
|    /     |      除      |      9 / 9      |      1       |
|    %     | 取模（取余） |     11 % 9      |      2       |
| （前）++ | 先运算后取值 |  a = 2;b = ++a  | a = 3;b = 3  |
| ++（后） | 先取值后运算 |  a = 2;b = a++  | a = 3;b = 2  |
| （前）-- | 先运算后取值 |  a = 2;b = --a  | a = 1;b = 1  |
| --（后） | 先取值后运算 |  a = 2;b = a--  | a = 1;b = 2  |
